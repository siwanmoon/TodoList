# TodoList

## 프로그램 설명

- 이 프로그램은 스프링 부트를 기반으로 한 웹 애플리케이션입니다


- 기능


- Create
  - 할 일 추가
  - 내용이 비어있거나 100자를 초과하면 오류 메시지가 표시됩니다
  - 최대 100개까지만 할 일을 등록할 수 있습니다


- Read
  - 할 일 조회
  - 마감 기한이 지났거나, 오늘 또는 내일 마감인 항목은 붉은색으로 표시됩니다
  - 중요 표시된 항목은 별표 표시로 강조됩니다
  - 완료된 항목은 취소선으로 표시됩니다


- Update
  - 완료 버튼으로 완료/미완료 상태를 변경할 수 있습니다
  - 중요 버튼으로 중요 상태를 변경할 수 있습니다
  - 수정 버튼으로 수정페이지로 이동하여 할일의 내용과 기한을 변경할 수 있습니다


- Delete
  - 삭제 버튼으로 할 일을 삭제할 수 있습니다

## 클래스 설명

### [common]

- `ErrorMessage` : 에러메세지를 모아놓은 상수 클래스
- `Strategy` : 프로그램 규칙에 관한 상수를 모아놓은 클래스


### [controller]

- `TodoController` : 프로그램을 관리하는 클래스


### [model]

- `Todo` : 할일 객체
- `TodoRepository` : `Todo`의 일급 컬렉션 객체


### [service]

- `TodoService` : TodoList의 서비스 인터페이스
- `TodoServiceImpl` : `TodoService`의 구현 객체


`Application` : 프로그램을 실행하는 객체


### [resources]

- `index.html` : 프로그램의 할 일 리스트 화면을 구현한 html
- `edit.html` : 할 일을 수정할때 사용하는 페이지를 구현한 html
